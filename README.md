# Tokenized Creator Studio

Title and thumbnail generator for the Tokenized Podcast.

## Features

- **Title Generator**: Paste transcripts or upload files (.txt, .srt, .vtt) to generate optimized YouTube titles
- **A/B Testing**: Select title variants for YouTube A/B tests
- **Thumbnail Concepts**: Preview thumbnail styles with custom text
- **Insights**: View top-performing title patterns based on Tokenized data

## Episode Types Supported

- Interview
- News & Analysis
- Deep Dive
- Predictions
- Stablecoin Stories (series)
- Stablecoin Stats (series)
- Agentic Commerce (series)
- Bonus / Special

## Deployment

Deployed on Vercel. Push to `main` to deploy.

## Local Development

Just open `index.html` in a browser - no build step required.
