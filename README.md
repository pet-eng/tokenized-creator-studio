# Tokenized Creator Studio

AI-powered title and thumbnail generator for the Tokenized Podcast.

## Features

- **AI Title Generator**: Uses Claude API to generate optimized YouTube titles based on your transcript
- **A/B Testing**: Select title variants for YouTube A/B tests
- **Thumbnail Concepts**: Preview thumbnail styles with custom text
- **Insights**: View top-performing title patterns based on Tokenized data

## Setup

### Environment Variables

Add to Vercel:
- `ANTHROPIC_API_KEY` - Your Anthropic API key from console.anthropic.com

## Deployment

Deployed on Vercel. Push to `main` to deploy.
